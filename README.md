# 💱 Currency Converter

A simple and responsive web-based **Currency Converter** that allows users to convert between global currencies in real-time using the [ExchangeRate API](https://www.exchangerate-api.com/).

## 🚀 Features

- 🔄 Real-time exchange rates
- 🔻 Dynamic dropdowns for currency selection
- 🔢 Converts between any two global currencies
- 🎨 Clean, modern UI with glassmorphism and smooth animations
- 🧠 Input validation for accurate results

---

## 📂 Project Structure
📁 currency-converter/
├── index.html # Main HTML structure
├── style.css # All styling and UI design
└── script.js # API calls and conversion logic


---

## 🛠️ How It Works

1. On page load, the app fetches the latest exchange rates from the ExchangeRate API.
2. All supported currencies are added dynamically to the dropdowns.
3. Users input an amount and select currencies to convert from and to.
4. Clicking **Convert** calculates and displays the converted amount instantly.
